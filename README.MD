# LeJOS Maven Tools
 maven tools for lejos(www.lejos.org)
 
 用于LeJOS开发的Maven工具包
 
## lejos-helloworld 
一个示例项目，演示了一个空的LeJOS项目的基本元素。

## lejos-helloworld-archetype
用于生成HelloWorld的原型，可以安装到本地仓库，并加入ArchetypeCatalog中方便使用。

## lejos-maven-plugin
用于执行ev3scpupload命令的插件，同样可以安装到本地仓库，它可以导入生成的jar文件到EV3设备中。
